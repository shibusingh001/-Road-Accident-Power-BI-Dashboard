# 🚦Road Accident Power BI Dashboard
An interactive Power BI dashboard designed to analyze and visualize road accident data, providing insights into casualty trends, accident severity, road conditions, and urban vs rural distribution.  This project helps identify key patterns and risk factors contributing to accidents, enabling data-driven decisions for improving road safety.
## 📌 Objective
The objective of this project is to analyze road accident data and uncover meaningful insights related to casualties, accident severity, road conditions, and environmental factors. The dashboard aims to support data-driven decision-making for improving road safety and reducing accident rates.
## 📊 Dataset Description
The dataset contains detailed information about road accidents, including:

Accident severity (Fatal, Serious, Slight)
Number of casualties
Vehicle types involved
Road type and surface conditions
Light conditions (Daylight/Darkness)
Urban and rural distribution
Time-based data (Months & Years: 2021–2022)

## 📈 Key KPIs

- 🚑 **Total Casualties:** 409K+
- 🚗 **Casualties by Vehicle Type**
- 🌆 **Urban vs Rural Accident Distribution**
- 🌙 **Impact of Light Conditions**
- 🛣️ **Road Type Analysis**
- 🌧️ **Road Surface Conditions (Dry, Wet, Ice)**

## 📊 Key Insights
- 🚑 Total casualties exceed **409,000**, highlighting the severity of road accidents.
- 🚗 Detailed breakdown of casualties by **vehicle type**.
- 🌆 Comparative analysis of **urban vs rural accident distribution**.
- 🌙 Evaluation of accident trends based on **light conditions** (day/night).
- 🛣️ Insights into accidents across different **road types**.
- 🌧️ Analysis of **road surface conditions** such as dry, wet, and icy roads.
